# 🔒 Python File Hash Checker

A command-line File Hash Checker built with Python that generates cryptographic hashes for files.

The application allows users to generate **MD5**, **SHA-1**, or **SHA-256** hashes, making it useful for learning about file integrity verification and basic cybersecurity concepts.

---

## Features

- 📄 Generate hashes for any file
- 🔐 MD5 hashing
- 🔐 SHA-1 hashing
- 🔐 SHA-256 hashing
- 📂 Read files in binary mode
- ✅ Input validation
- ⚠️ Error handling for invalid file paths

---

## Technologies Used

- Python 3
- hashlib

---

## How to Run

Clone the repository:

```bash
git clone https://github.com/TheAbdullahSaleem/python-file-hash-checker.git
```

Run the program:

```bash
python main.py
```
